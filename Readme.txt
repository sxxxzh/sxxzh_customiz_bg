sxxxzh自定义背景工具使用说明

基本介绍
sxxxzh自定义背景工具是一款轻量级桌面美化工具，可以为特定应用程序窗口添加自定义背景图片，并支持多种图像效果调节。

运行方式
- 运行程序后，会在系统托盘区域显示图标
- 右键点击托盘图标可打开菜单

托盘菜单功能
- 退出程序 - 完全关闭应用程序
- 设置开机自启 - 配置程序随系统启动  
- 隐藏托盘 - 隐藏系统托盘图标

配置文件说明
配置文件采用JSON格式，保存在config.json文件中

主要参数：

基本设置
enabled: true/false          总开关
scan_interval: 3             扫描间隔(秒)，值越小响应越快，CPU占用越高

目标程序设置（可以配置多个程序）
name: "程序名称"             程序标识名称
keywords: ["关键字"]         识别程序窗口的关键字，如进程名、窗口标题
image_path: "图片路径"       背景图片路径，支持相对路径
alpha: 40                    透明度，0-100，值越小越透明
brightness: 1.0              亮度调节，1.0为原始亮度
contrast: 1.0                对比度调节，1.0为原始对比度
saturation: 1.0              饱和度调节，1.0为原始饱和度

完整配置示例：
{
    "enabled": true,
    "scan_interval": 3,
    "targets": [
        {
            "name": "Weixin",
            "keywords": ["Weixin.exe", "Weixin"],
            "image_path": "background.png",
            "alpha": 40,
            "brightness": 1.0,
            "contrast": 1.0,
            "saturation": 1.0
        }
    ]
}

使用注意事项
1. 确保背景图片文件存在且路径正确
2. 修改配置文件后需要重启程序生效
3. 扫描间隔建议设置在3-5秒，平衡性能和响应速度
4. 可以为不同的程序配置不同的背景效果
5. 某些安全软件可能会误报，如需使用请添加信任

技术支持
如遇问题请检查：
- 配置文件格式是否正确
- 背景图片文件是否存在
- 目标程序是否正在运行
- 有些程序由于渲染技术不能正常挂载背景 现已知如下：Steam ，Typora ，windows 终端 等
